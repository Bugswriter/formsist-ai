Formsist-AI is a smart assistant that uses Google Gemini and LangChain to understand and auto-fill HTML forms based on your personal portfolio. It includes a command-line interface (CLI) and a planned browser extension.

üîß Backend CLI

To set up and use the CLI tool:

git clone https://github.com/bugswriter/formsist-ai
cd formsist-ai/backend

Create a virtual environment and install dependencies:

python -m venv env
source env/bin/activate  # On Windows: .\env\Scripts\activate
pip install -r requirements.txt

Set your Gemini API key:

# Linux/macOS
export GOOGLE_API_KEY="YOUR_KEY"

# Windows CMD
set GOOGLE_API_KEY="YOUR_KEY"

# PowerShell
$env:GOOGLE_API_KEY="YOUR_KEY"

Create a portfolio.txt file with your information. Then run:

python ask.py portfolio.txt "What's my email address?"
